Notification rules

Automatic checks run every three hours. Notifications are sent only when a meaningful forecast change or new NWS alert is detected.

  • Temperature changes by 5°F or more
  • Rain probability changes by 20 points or more
  • Wind-gust forecast changes by 10 mph or more
  • Condition category changes
  • New Delaware watches, warnings, or advisories
