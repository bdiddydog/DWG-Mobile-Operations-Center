Netlify environment variables

Add these four variables in Netlify under Project configuration → Environment variables.

NameValue
RAINBOW_API_KEYYour Rainbow Weather API key
VAPID_PUBLIC_KEYYour generated public VAPID key
VAPID_PRIVATE_KEYYour generated private VAPID key
VAPID_SUBJECTmailto:your-email@example.com

You do not edit app.js. The app reads the public key from Netlify automatically.

Deployment requirement

Push notifications require Netlify Functions. A simple static Netlify Drop may show the page but will not run the secure weather and push-notification backend.

Use one of these:

  1. Connect the folder to GitHub and let Netlify build it, or
  2. Deploy from the Linux terminal using Netlify CLI.

Phone installation

  1. Open the deployed site on your phone.
  2. Add it to your home screen.
  3. Open it from the new icon.
  4. Tap Enable phone alerts.
  5. Tap Send test alert.
